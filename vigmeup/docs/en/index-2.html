2. Installation

2.1 How to obtain VigMeUp

You can obtain VigMeUp from ftp.kde.org, metlab.unc.edu, their respective mirrors, and of course, the main site at http://www.c-nix.com/vigmeup RPMS and Source RPMS for RedHat and SuSE as well as Slackware packages are available.

2.2 Requirements

You need at least KDE 1.0. I think you might need KDE 1.1.1, but I'm not sure (sorry). If VigMeUp 1.0.2 doesn't work for you, use VigMeUp 0.1.1 for KDE 1.0 and VigMeUp 0.7b for KDE Beta4.

2.3 Compilation and installation

In order to compile and install VigMeUp on your system, type the following in the base directory of the VigMeUp distribution: 

% ./configure
% make
% make install

Since VigMeUp uses 

autoconf
you should have not trouble compiling it. Should you run into problems please report them to the the author at Sean Melody
